# GeoPhoto
<p>Application allow user to make pictures of interesting places. Then show them at google maps where pictures was made.
Made for Android programming course by Allegro Group at Adam Mickiewicz University 

Technology used:
Firebase,
MVP Pattern,
Google Maps Api,
ButterKnife
</p>
